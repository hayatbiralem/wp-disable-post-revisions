A simple lightweight WordPress plugin to disable post revisions.

== Description ==
This plugin allows you to disable post revisions for all post types just by enable but if you want more control over the post revision counts of each post types, you can do it too, easily.

== Installation ==
1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the \'Plugins\' menu in WordPress
3. Place `` in your templates
